# My Todo-List

A modern **To-Do List Application** built with **React + Vite + Tailwind CSS**.
This app allows users to manage daily tasks efficiently with a clean Glass UI design.

---

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Responsive UI
* Modern Glassmorphism design
* Gradient and animated UI effects

---

## Tech Stack

* React 19
* Vite
* Tailwind CSS v4
* React Icons
* ESLint

---

## Getting Started

Follow the steps below to run the application locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Gokulmlk/Todo_App_Assign.git
```

### 2. Navigate to the Project Folder

```bash
cd Todo_App_Assign
```

### 3. Install Dependencies

Make sure you have **Node.js (v18 or above)** installed.

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

After running the command, open the URL shown in the terminal (usually):

```
http://localhost:5173

```
